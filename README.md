## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Run Frontend

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Run Docker Container of Redis and Postgresql 
```bash
  docker compose up
```

Run backend

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```
